# Section 0 - Mathematical Foundations
## 3. Proportionality

We start with the Universal Law of Gravitation:

F = (G m1 m2) / r^2

where:

F = gravitational force  
G = gravitational constant  
m1 and m2 = masses  
r = distance between the masses

We need to determine how the force changes if:

- the distance r is doubled
- both masses m1 and m2 are halved

---

## Step 1: Original force

The original force is:

F = (G m1 m2) / r^2

---

## Step 2: Apply the changes

If the masses are halved:

m1 → m1 / 2  
m2 → m2 / 2

If the distance is doubled:

r → 2r

Substitute these values into the formula.

---

## Step 3: New force expression

F_new = G (m1/2)(m2/2) / (2r)^2

---

## Step 4: Simplify the expression

First simplify the numerator:

(m1/2)(m2/2) = m1m2 / 4

So:

F_new = G (m1m2 / 4) / (2r)^2

Now simplify the denominator:

(2r)^2 = 4r^2

So:

F_new = (G m1 m2 / 4) / (4r^2)

---

## Step 5: Final simplification

F_new = (G m1 m2) / (16 r^2)

---

## Step 6: Compare with the original force

Original force:

F = (G m1 m2) / r^2

New force:

F_new = (G m1 m2) / (16 r^2)

So:

F_new = F / 16

---

## Final Result

The gravitational force becomes **1/16 of the original force**.

So the force decreases by a factor of **16**.

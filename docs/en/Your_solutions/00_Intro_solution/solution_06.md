## 6. Function Analysis

Consider the function:

f(x) = 3x^2 - 12x + 7

We need to identify any local maxima or minima.

---

## Step 1: Find the first derivative

f'(x) = d/dx (3x^2 - 12x + 7)

f'(x) = 6x - 12

---

## Step 2: Find critical points

Critical points occur where:

f'(x) = 0

So:

6x - 12 = 0

6x = 12

x = 2

---

## Step 3: Determine if it is a maximum or minimum

Find the second derivative.

f''(x) = d/dx (6x - 12)

f''(x) = 6

Since:

f''(x) > 0

the function has a **local minimum** at x = 2.

---

## Step 4: Find the function value

f(2) = 3(2^2) - 12(2) + 7

f(2) = 12 - 24 + 7

f(2) = -5

---

## Final Result

The function has a **local minimum** at:

x = 2

The minimum value is:

f(2) = -5

So the local minimum point is:

(2, -5)

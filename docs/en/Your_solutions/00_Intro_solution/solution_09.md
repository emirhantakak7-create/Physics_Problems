# Section 0 - Mathematical Foundations
## 9. Optimization Problem

A rectangle is placed under the curve:

y = 3 - x^2

in the first quadrant.

We need to find the dimensions of the rectangle with the maximum area.

---

## Step 1: Express the area in terms of x

Since the rectangle is in the first quadrant and its upper-right corner lies on the curve, its:

width = x  
height = y = 3 - x^2

So the area is:

A(x) = x(3 - x^2)

A(x) = 3x - x^3

---

## Step 2: Find the derivative of the area function

A'(x) = d/dx (3x - x^3)

A'(x) = 3 - 3x^2

---

## Step 3: Find critical points

Set the derivative equal to zero:

3 - 3x^2 = 0

3x^2 = 3

x^2 = 1

x = 1

Since we are in the first quadrant, we take:

x = 1

---

## Step 4: Find the corresponding height

Substitute x = 1 into the curve:

y = 3 - x^2

y = 3 - 1^2

y = 2

---

## Step 5: Verify that it gives a maximum

Find the second derivative:

A''(x) = d/dx (3 - 3x^2)

A''(x) = -6x

At x = 1:

A''(1) = -6 < 0

So the area is maximum at x = 1.

---

## Final Result

The rectangle with maximum area has:

width = 1  
height = 2

So the dimensions are:

1 × 2

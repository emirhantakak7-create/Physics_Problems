# 2. Range Optimization

For projectile motion, the range is given by:

R(θ) = (v0^2 sin(2θ)) / g

We need to show analytically that the maximum range is achieved when the launch angle is 45°.

---

## Step 1: Identify the variable part

In the formula

R(θ) = (v0^2 sin(2θ)) / g

the quantities v0 and g are constants for a given projectile.

So the only part that changes with θ is:

sin(2θ)

Therefore, maximizing R(θ) is the same as maximizing sin(2θ).

---

## Step 2: Use the maximum value of sine

The sine function satisfies:

-1 ≤ sin(2θ) ≤ 1

Its maximum possible value is:

sin(2θ) = 1

This happens when:

2θ = 90°

So:

θ = 45°

---

## Step 3: Substitute into the range formula

When θ = 45°, we have:

sin(2 × 45°) = sin(90°) = 1

So the range becomes:

R_max = (v0^2 × 1) / g

R_max = v0^2 / g

---

## Final Result

The range is maximum when:

θ = 45°

and the maximum range is:

R_max = v0^2 / g
